Die Angaben sind anonym. Dafür benutzen wir nicht den jeweiligen Namen, sondern einen „Code“.

Bitten den Code folgendermaßen bilden:

  • die ersten beiden Buchstaben des Vornamens der Mutter,
  • den Tag des Geburtsdatums
  • und die ersten beiden Buchstaben des Vornamens.

    • Beispiel: AN08PE für Anna (Vorname der Mutter), 08. Januar 1999 (eigenes Geburtsdatum), Peter (eigener Name)

    Bitte den eigenen Code nach obigem Muster in das Textfeld eintragen.
